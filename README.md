# Quiz App

> This a Note App Built with reactjs and typescript where you can keep notes of your daily activities.


## Built With

- JavaScript
- TypeScript
- HTML5
- Bootstrap
- CSS

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

Text editor
Get a browser like chrome or firefox in their recent versions
Node.js
Node Package Manager

### Setup

- Install required dependencies `npm install`
- Run `npm start` to start the local webserver

### Usage

- Clone this Repository using the command `git@github.com:keneogu/Noteapp-Typescript.git`
- If you wish to play around the codes, navigate to this Repository in the Text Editor
